# Oral Cancer Detection Using Deep Learning

## Overview
Engineered a deep learning image classification system to detect early-stage oral cancer 
from clinical photographs using CNN architectures — published as peer-reviewed research 
under faculty supervision at Panimalar Engineering College, 2024.

## Technologies Used
- Python
- TensorFlow
- CNN (ResNet, VGG)
- Image Processing
- Scikit-learn

## Key Contributions
- Built CNN-based classification model using ResNet and VGG architectures
- Applied advanced image preprocessing techniques on clinical photographs
- Achieved significant improvement in classification accuracy over traditional methods
- Documented and submitted findings for peer-reviewed publication

## Results
- Published peer-reviewed research paper — Panimalar Engineering College, 2024
- Demonstrated AI application in real-world healthcare diagnostics
